# Solarled Landing

Landing page premium para Solarled, S.A. construida con:

- Next.js App Router
- TypeScript
- Tailwind CSS
- Framer Motion

## Cómo correr localmente

```bash
npm install
npm run dev
```

## Build de producción

```bash
npm install
npm run build
npm run start
```

## Notas del proyecto

- La landing está pensada para GitHub Web + Vercel.
- El formulario envía la información a WhatsApp con mensaje prellenado.
- El correo también queda disponible como salida secundaria.
- Como no se recibió un logo limpio/vectorial, se creó un wordmark visual temporal dentro del código.
- La cuenta bancaria no se muestra en la web pública.

# Deploy en Vercel

1. Sube la carpeta del proyecto a un repositorio en GitHub.
2. Entra a Vercel.
3. Haz clic en **Add New > Project**.
4. Conecta el repositorio.
5. Vercel detectará Next.js automáticamente.
6. Deja la configuración por defecto.
7. Haz clic en **Deploy**.

No requiere variables de entorno para esta versión.
